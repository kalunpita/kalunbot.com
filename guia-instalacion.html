Guía de Instalación

Configuración paso a paso de KalunBot

← Volver al inicio
⚠️ Importante: Lea completamente el disclaimer antes de instalar. El uso de este software puede resultar en la suspensión de su cuenta de EA FC 25.

Requisitos del Sistema

Requisitos Mínimos:

  • OS: Windows 10/11 (64-bit)
  • RAM: 4 GB mínimo, 8 GB recomendado
  • Resolución: 1920x1080 (Full HD) requerida
  • Conexión: Internet estable
  • EA FC 25: Instalado y funcionando
  • Plataforma: PC (Steam/EA App/Epic Games)

Software Adicional:

  • Python 3.8+ (se instala automáticamente si no está presente)
  • Microsoft Visual C++ Redistributable
  • Google Chrome o Edge (para web automation)

Instalación

1 Descargar KalunBot

Descargue la última versión desde nuestro sitio oficial o GitHub:

  • Archivo: KalunBot-Setup-v1.0.exe
  • Tamaño: ~25 MB
  • Verificar hash SHA256 (recomendado)
2 Ejecutar el Instalador

Haga clic derecho en el archivo descargado y seleccione "Ejecutar como administrador":

  • Windows puede mostrar una advertencia de seguridad - esto es normal
  • Haga clic en "Más información" → "Ejecutar de todas formas"
  • Siga las instrucciones del instalador
3 Seleccionar Directorio de Instalación

Recomendamos instalar en la ubicación predeterminada:

  • Ubicación: C:\Program Files\KalunBot\
  • Asegúrese de tener permisos de escritura
  • Evite rutas con caracteres especiales o espacios
4 Instalación de Dependencias

El instalador descargará automáticamente las dependencias necesarias:

  • Python y librerías requeridas
  • Drivers de automatización web
  • Archivos de configuración
Nota: Este paso requiere conexión a internet y puede tardar varios minutos.

Primera Configuración

5 Configuración de Pantalla

KalunBot requiere una configuración específica de pantalla:

  • Resolución: 1920x1080 (requerida)
  • Escalado: 100% (sin escalado DPI)
  • Monitor principal si tiene múltiples pantallas

Cómo configurar la resolución:

  1. Clic derecho en el escritorio → "Configuración de pantalla"
  2. Cambiar resolución a 1920x1080
  3. Establecer escalado al 100%
  4. Aplicar cambios y reiniciar si es necesario
6 Configuración de EA FC 25

Configure EA FC 25 en modo ventana para mejor compatibilidad:

  • Abrir EA FC 25
  • Ir a Configuración → Pantalla
  • Modo: Ventana sin bordes o Ventana
  • Resolución: 1920x1080
7 Primer Inicio de KalunBot

Inicie KalunBot desde el escritorio o menú inicio:

  • El programa se iniciará con la configuración por defecto
  • Aparecerá el asistente de configuración inicial
  • Complete los campos básicos (nombre de jugador, preferencias)

Configuración Avanzada

8 Configurar Parámetros de Sniping

Ajuste los parámetros según sus necesidades:

  • Modo Sport: Para búsquedas rápidas (recomendado)
  • Precio máximo: Establecer límite de compra
  • Delay entre búsquedas: 0.01s para modo sport
  • Límite de compras: Número máximo de cartas a comprar
9 Test de Funcionamiento

Realice una prueba para verificar que todo funciona correctamente:

  1. Abra EA FC 25 y vaya al mercado de transferencias
  2. Configure una búsqueda básica en KalunBot
  3. Inicie el bot en modo de prueba
  4. Verifique que el bot navega correctamente por el menú

Solución de Problemas Comunes

❌ "No se puede encontrar EA FC 25"

  • Asegúrese de que EA FC 25 esté abierto y en el menú principal
  • Verifique que la resolución sea exactamente 1920x1080
  • Reinicie tanto EA FC 25 como KalunBot

❌ "Error de dependencias de Python"

  • Ejecute KalunBot como administrador
  • Reinstale usando el instalador oficial
  • Verifique que tiene conexión a internet durante la instalación

❌ "El bot no hace clic correctamente"

  • Verifique el escalado DPI (debe estar al 100%)
  • Desactive el modo de compatibilidad de Windows
  • Asegúrese de que no hay ventanas superpuestas

❌ "Detección de captcha/antifraude"

  • Reduzca la velocidad del bot
  • Use pausas más largas entre acciones
  • Evite usar el bot por períodos muy largos

Recomendaciones de Uso

💡 Consejos para un Uso Seguro:

  • Use cuentas secundarias para pruebas
  • No deje el bot funcionando toda la noche
  • Haga pausas regulares (30-60 minutos de uso, 15-30 min de pausa)
  • Mantenga actualizado tanto EA FC 25 como KalunBot
  • Monitoree regularmente el estado de su cuenta

Actualización

KalunBot verificará automáticamente las actualizaciones al iniciar. Para actualizar manualmente:

  1. Descargue la nueva versión del sitio oficial
  2. Cierre KalunBot completamente
  3. Ejecute el nuevo instalador
  4. Sus configuraciones se conservarán automáticamente

Desinstalación

Para desinstalar KalunBot completamente:

  1. Panel de Control → Programas → Desinstalar un programa
  2. Seleccione "KalunBot" → Desinstalar
  3. Elimine manualmente la carpeta de configuración si lo desea
  4. Se abrirá automáticamente TrustPilot para dejar una reseña
